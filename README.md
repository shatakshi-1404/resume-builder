# ResumeForge — Free Resume Builder

A clean, free, browser-based resume builder with live preview and one-click PDF export. No sign-up, no backend, no data stored anywhere.

## Live Demo
[resume-builder-xyz.vercel.app](https://resume-builder-mocha-eta.vercel.app/) ← replace with your Vercel URL

## Features
- Live A4 resume preview as you type
- Sections for Personal Info, Summary, Education, Experience, Projects, Skills, and Certifications
- Add / remove multiple entries per section
- One-click PDF download via browser print
- ATS-friendly layout with clean serif typography
- Zero dependencies — plain HTML, CSS, and JavaScript

## Built With
- HTML / CSS / JavaScript (no framework, no build step)
- Deployed on Vercel free tier

## How to Run Locally
```bash
git clone https://github.com/shatakshi-1404/resume-builder.git
cd resume-builder
# open index.html in your browser — done
```
